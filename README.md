# AI Surveillance Platform

## Features
- Face Recognition
- Unknown Person Detection
- Zone Intrusion Detection
- Loitering Detection
- Risk Scoring
- Real-time Dashboard
- PDF Report Generation
- Live Event Monitoring

## Tech Stack
Backend:
- FastAPI
- SQLAlchemy
- SQLite
- OpenCV
- YOLOv8

Frontend:
- React
- TypeScript
- Recharts

## Run Backend

cd backend
source venv/bin/activate
uvicorn app.main:app --reload

## Run Frontend

cd frontend
npm install
npm run dev
